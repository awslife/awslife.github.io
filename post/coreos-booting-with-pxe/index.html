CoreOS booting with PXE
Kubernetes CoreOS Container OS VMware Linux PXE
Lastmod: 2019-10-24

CoreOS Cluster 구성 방법을 정리하다 PXE를 통한 부팅이 필요하여 구성 방법을 정리하였다. 모든 환경은 VMware 위에서 테스트되었다.

요구사항

PXE를 통한 원격 부팅을 하기 위해서는 DHCP, TFTP, PXE 서버가 필요하다. 각각의 서비스를 구성하는 방법을 모두 정리하기에는 너무 방대한 양이므로 여기에서는 구성 방법을 모두 생략하고 PXE 부팅 설정에만 집중하였다.

pxelinux 구성

CoreOS에서 제공하는 pxe 옵션은 8개가 있으며 각 옵션에 대한 자세한 정보는 CoreOS 홈페이지에서 확인 가능하다.

  • rootfstype=tmpfs : 쓰기 가능한 루트 파일 시스템에 tmpfs를 사용
  • root=/dev/sda1, root=LABEL=ROOT, root=UUID=2c618316-d17a-4688-b43b-aa19d97ea821 처럼 표현 가능
  • sshkey : 지정한 공개키를 core 사용자의 authrized_keys 파일에 추가 가능
  • console : tty에서 커널 출력과 로그인 프롬프트를 활성화. 기본값은 tty0. console=tty0, console=ttyS0 처럼 여러번 지정 가능
  • coreos.autologin : 패스워드 프롬프트 없이 쉘로 진입, 이 옵션을 사용할 경우 console 옵션과 결합하여 사용해야 함
  • coreos.first_boot=1: ignition 구성을 다운로드하고 이를 통하여 부팅.
  • coreos.config.url : 지정된 url에서 ignition 구성을 다운로드 한다. http, https, s3 및 tftp 방식 지원 가능
  • ip : initramfs에 대한 임시 정적 네트워킹을 구성한다. dhcp가 없는 환경에서 최초 부트 프로비저닝에 유용하다.

PXE 부팅을 위해 아래와 같은 트리를 구성하였다.

awslife@k8sdev:~/Downloads/pxe$ tree pxe

default 파일

PXE 부팅의 경우 GUID, MAC, IP 및 Default 순으로 부팅을 시도하며 세부 구성을 하지 않을 경우 마지막 순서인 default 파일을 읽어 부팅을 시도한다. 자세한 정보는 아래 Reference Url의 PXELINUX 링크에서 확인 가능하다. default에서는 coreos의 필수 필요 속성만을 설정하여 부팅되도록 설정하였다. 세부 정보는 아래와 같다.

awslife@k8sdev:~/Downloads/pxe$ cat pxelinux.cfg/default
default coreos
prompt 5
timeout 15

display boot.msg

label coreos
  menu  label ^CoreOS
  kernel coreos/coreos_production_pxe.vmlinuz
  initrd coreos/coreos_production_pxe_image.cpio.gz
  append coreos.first_boot=1 coreos.config.url=tftp://192.168.11.253/coreos/coreos-config.ign
awslife@k8sdev:~/Downloads/pxe$

OEM 구성

CoreOS는 VMware 상에서 테스트되었기 때문에 CoreOS 내부에 vmtools 설치가 필요하다. CoreOS에서는 이런 경우를 위해 oem 환경을 구성할 수 있다. 이에 대한 세부 정보는 Reference URL의 Notes for distributors와 Running CoreOS Container Linux on VMware 항목을 참조하기 바란다.

OEM 항목이 구성 완료되면 PXE 부팅시 반영은 매우 간단하다. initrd 마지막에 oem 관련 내용만 추가해주면 된다. oem.cpio.gz이 추가된걸 확인할 수 있다.

awslife@k8sdev:~/Downloads/pxe$ cat pxelinux.cfg/default
default coreos
prompt 5
timeout 15

display boot.msg

label coreos
  menu  label ^CoreOS
  kernel coreos/coreos_production_pxe.vmlinuz
  initrd coreos/coreos_production_pxe_image.cpio.gz,coreos/oem.cpio.gz
  append coreos.first_boot=1 coreos.config.url=tftp://192.168.11.253/coreos/coreos-config.ign
awslife@k8sdev:~/Downloads/pxe$
